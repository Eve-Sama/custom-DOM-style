### ☀️ 介绍

本项目是一个浏览器插件, 支持Chrome(Firefox暂时有点兼容性问题, 暂时没时间解决). 技术栈为JQuery+TypeScript+Gulp.

### How to run

clone项目后, 先执行npm run star(必须), 稍等几秒后, 即可导入浏览器, 插件生效. 在完成插件开发之前, 不建议使用该插件.
